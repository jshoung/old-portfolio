ContactHub

About this project

ContactHub is a desktop Address Book application, designed for NUS students, I worked on as part of my NUS coursework.

I functioned as the Team Lead and oversaw overall project coordination.

Our team comprised of 4 members and collaborated using GitHub (Forking Workflow). You can view the repo and download the app here!

Note: This application is not actively maintained so certain features may no longer work.

  • Created with Java, built with Gradle, tested with Travis CI
  • Features complete user and developer documentation
