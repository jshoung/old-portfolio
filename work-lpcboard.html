LPC1769 LPCXpresso

About this project

I worked with the LPC1769 LPCXpresso Board as part of my NUS coursework. This module taught me about microprocessor/System-on-Chip(SoC) concepts and how to use them to interface with the external world. I gained a lot of experience in reading data-sheets, using libraries etc.

The project involved creating a simulation of a manned space flight system called NUSpace. I utilized various devices on the board such as the accelerometer, light and temperature sensor to implement this. The XBee RF module was used to communicate with Tera Term terminal on the computer.

  • Worked with ARM Assembly and C programming langauges
  • LPC1769 LPCXpresso Board with NXP's ARM Cortex-M3
